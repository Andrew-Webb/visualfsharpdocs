# Seq.delay<'T> Function (F#)

Returns a sequence that is built from the given delayed specification of a sequence.

**Namespace/Module Path**: Microsoft.FSharp.Collections.Seq

**Assembly**: FSharp.Core (in FSharp.Core.dll)
